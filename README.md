# 🌿 나만의 감성 다이어트 정원 (Kcal-Garden)
Zero UI와 감성적인 내러티브를 결합한 AI 기반 식단 다이어리 앱입니다. 
숫자에 얽매이는 다이어트의 스트레스를 줄이고, 나만의 정원을 가꾸는 따뜻한 경험을 제공합니다.

## ✨ 주요 기능
- **AI 식단 분석**: 줄글로 편하게 입력하면 Gemini AI가 칼로리를 자동 계산합니다.
- **감성 멘탈 케어**: 그날의 식단과 감정에 맞춰 AI가 다정한 위로와 코멘트를 건넵니다.
- **나만의 정원 가꾸기**: 목표 칼로리 달성률에 따라 씨앗이 싹을 틔우고 예쁜 꽃으로 피어납니다.
- **오프라인 저장**: DataStore를 활용해 앱을 껐다 켜도 나의 소중한 기록이 유지됩니다.

## 🛠 기술 스택
- **Language**: Kotlin
- **UI Toolkit**: Jetpack Compose
- **AI Integration**: Google Generative AI SDK (Gemini 2.5 Flash)
- **Local Storage**: Preferences DataStore, Gson

## 🚀 직접 실행해보기
1. 우측의 **Releases** 탭에서 `Kcaltxt.apk` 파일을 다운로드하여 스마트폰에 설치해 보세요!
2. 코드를 직접 빌드하려면 `ApiKey.kt` 파일에 본인의 Gemini API Key를 입력해야 합니다.
